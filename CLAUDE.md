# 単語アプリ（WORDCRAFT / ターゲット1900）— Claude Code 作業規約

このリポジトリで Claude Code が作業する際のプロジェクト規約。
上位のローカル規約（`~/.claude/CLAUDE.md`・`◆AI関連/CLAUDE.md`）に従ったうえで本規約を適用する。

## アプリ概要
- 高校生向け英単語アプリ（旺文社ターゲット1900・全1900語）。完全オフライン PWA・単一HTML。
- 本番: https://genmaojun.github.io/tango1900/ （GitHub `genmaojun/tango1900`・main/root 配信）
- マインクラフト風デザイン（テーマ既定 `minecraft`）。

## ソースと生成（重要）
- **正は `_app_template.html`**（プレースホルダ `__WORDS_JSON__`）。`index.html` と `dist/` は **`py build.py` で生成される成果物**。`index.html` を手で直接編集しない。
- 更新手順: `_app_template.html` を編集 → `py build.py` → 生成物を確認。
- ⚠️ **CSS/JS/フォントを変えたら `build.py` の Service Worker CACHE 版（`tango1900-vNN`）を必ず +1 する。** 上げ忘れると、ホーム追加済みの既存 PWA が古いキャッシュのまま新版を受け取れない（過去にハマった要点）。フォント等の新規アセットを足したら SW の `ASSETS` と dist コピー一覧にも追記する。

## フォント方針（2026-06-23 確定）
- **学習で実際に読む文字＝可読なシステムフォント**：英単語(`.word`/`.word-sm`/`.mic-word`)・例文英語(`.example .en`)・日本語訳(`.meaning`/`.example .ja`/`.mic-ja`)。
- **それ以外の GUI（見出し・ボタン・選択肢 `.ans`・数字・ロゴ）＝ピクセルフォント維持**（`PixelMC`/`JpDot`）でマイクラ感を保つ。
- マイクラテーマ CSS は前方一致 `html[data-theme^="minecraft"]` で `minecraft`/`minecraft2` 両対応。`<button>` は標準でフォントを継承しないので、必要なフォントは明示指定する。

## デバッグ7視点（デフォルト・必須）
コード変更後は最低この7視点で確認する。規模に応じて増減してよいが、**「壊れて困る順（リスクの大きい順）」に潰す**。
1. **静的** … 構文・実行エラーが出ない
2. **機能・正常系** … 仕様どおり動く
3. **表示・UI** … フォント・レイアウト崩れなし
4. **異常系・境界値** … 0・最大・空・連打
5. **回帰** … 既存を壊していない・既存データ(localStorage)互換
6. **非機能** … 速度・オフライン・電池
7. **実機・環境差** … iPhone Safari・OS差（プレビューは Win/Chromium なので差が出る）

> 家族へ配布（LINE 周知）する前は、特に ⑦実機 iPhone 目視 ＋ ⑥オフライン ＋ ⑤記録互換 ＋ SW 版上げ を必ず確認する。
