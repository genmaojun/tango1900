# 元の英単語帳 ターゲット1900（スマホ単語帳アプリ）

高校1年・井上元くん用。通学電車（二子玉川⇄外苑前）で使う、**英単語ターゲット1900**（旺文社／全1900語）対応の単語帳アプリ。英検2級も視野。

## 何ができるか
- 英単語フラッシュカード（英→日／日→英）
- 🔊 **発音ボタン**（iPhone内蔵音声・オフライン再生、米／英なまり切替）
- **間違い記録**：✕にした単語が「苦手」に貯まり、何度でも復習できる
- 進捗の見える化（覚えた／苦手／未学習、セクション別バー）
- Part1〜3・100語ごとのセクション選択、英検2級モード（Part1・2）
- 設定（速度・自動発音・出題順・1回の語数・テーマ）
- **完全オフライン・1ファイル完結**（電波が切れる地下鉄でもOK）
- 学習記録はiPhone内（localStorage）に保存

## 配布物
- **`index.html`** … これ1つで動くアプリ本体（これをiPhoneで開く）

## iPhoneでの使い方
1. `index.html` をSafariで開く（ホスティング推奨）
2. 共有メニュー →「**ホーム画面に追加**」
3. ホームのアイコンから起動（オフラインで動作）
※初回だけ画面を1タップすると発音が有効化されます（イヤホン推奨）。

## 単語データの作り直し・更新（開発メモ）
元データは旺文社のターゲット1900単語リストPDF（`target1900_source.pdf`）。

```
py -3 extract_1900.py   # PDF → words_1900.json（全1900語抽出）
py -3 build.py          # _app_template.html + words_1900.json → index.html
```

- `_app_template.html` … アプリのUI/ロジック（プレースホルダ `__WORDS_JSON__` にデータを流し込む）
- `extract_1900.py` … pdfplumber でPDFの表を抽出
- `build.py` … テンプレへデータ注入して `index.html` を生成
- 必要ライブラリ: `py -3 -m pip install pdfplumber`
